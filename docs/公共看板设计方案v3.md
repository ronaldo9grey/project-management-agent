# 公共看板设计方案 v3.0

> 基于反馈优化：可视化图表替代文字，时间线展示任务节点

---

## 一、设计原则

### 核心原则

1. **可视化优先** — 一图胜千言，用图表替代表格
2. **信息分层** — 总览 → 项目详情 → 任务节点
3. **一目了然** — 3秒内把握全局，10秒内发现问题

### 视觉风格

- 使用进度条、环形图、柱状图等可视化组件
- 颜色编码：🟢健康 🟡关注 🔴风险
- 关键数字突出显示，次要信息折叠

## 三、可视化组件设计

### 3.1 环形进度图（健康度分布）

```typescript
// 使用 SVG 绘制
const DonutChart = ({ data }) => {
  const total = data.reduce((sum, d) => sum + d.value, 0)
  let offset = 0
  
  return (
    <svg viewBox="0 0 100 100">
      {data.map((d, i) => {
        const percent = (d.value / total) * 100
        const dash = (percent * 251.2) / 100
        return (
          <circle
            key={i}
            cx="50" cy="50" r="40"
            fill="none"
            stroke={d.color}
            strokeWidth="20"
            strokeDasharray={`${dash} 251.2`}
            strokeDashoffset={-offset * 2.512}
          />
        )
        offset += percent
      })}
    </svg>
  )
}
```

## 五、交互设计

### 5.1 点击展开

- 点击时间线节点 → 展开项目详情卡片
- 点击进度条 → 跳转到项目详情页

### 5.2 筛选

- 顶部筛选器：按状态筛选（全部/健康/风险/延期）
- 时间线：可拖动查看不同时间段

### 5.3 实时更新

- 预警数字实时更新（WebSocket 或轮询）
- AI洞察按需刷新按钮

## 六、技术选型

| 组件 | 方案 | 原因 |
|------|------|------|
| 环形图 | SVG | 纯CSS/SVG，无需依赖库 |
| 柱状图 | Chart.js | 轻量，支持响应式 |
| 时间线 | 自定义CSS | 灵活控制，易于定制 |
| 进度条 | CSS | 原生实现，性能好 |

## 七、对比

### 修改前 vs 修改后

| 维度 | 修改前 | 修改后 |
|------|--------|--------|
| 信息密度 | 高（大量文字） | 低（图表为主） |
| 健康度展示 | 排名列表 + 项目全景重复 | 环形图 + 进度条 |
| 任务节点 | 无 | 时间线展示 |
| 可视化 | 少 | 多 |
| 理解时间 | 需要阅读 | 3秒内把握全局 |
